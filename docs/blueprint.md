# **App Name**: Altitude Portfolio

## Core Features:

- Landing Page Selector: Dynamically direct users (HR, Data Professional, Stalker, Hiring Manager) to custom portfolio views based on their identified role.
- Projects and Blogs: Dedicated pages displaying specific projects and blog content catered to each identified user type. Example: HR role shows Certificates page, but the stalker persona is displayed the Blogs posts.
- Skills Radar Chart: Interactive radar chart displaying technical proficiency across multiple skills with smooth transitions and hover effects.
- 3D Tilt Cards: Project cards with 3D tilt effect based on mouse movement, enhanced with hover effects and shadows for visual depth.
- Project Detail Modal: Modal popup with detailed project information (achievements, duration, team size, tech stack) upon card click.
- AI Portfolio Assistant: An AI-powered portfolio assistant in the bottom-left corner to answer questions about your skills, projects, and certifications. The AI uses a tool to provide context-aware answers, quick question shortcuts, typing indicators and smooth animations. This can be minimized/maximized by the user.
- Leadership and Accolades Tab: New tab showcasing leadership experience and awards with grid layout, animated gradient headers, and expanding modals for details.
- Smart Content Distribution: The user (HR, Data Professional, Stalker, Hiring Manager) view shows a certificate gallery, code samples, interactive analytics or other tailored content

## Style Guidelines:

- Primary color: Sky blue (#4DB8FF) for a sense of aviation and forward movement.
- Background color: Light gray (#F0F4F7), offering a clean, professional backdrop that is easy on the eyes.
- Accent color: Electric blue (#7DF9FF), providing visual interest and highlights.
- Body font: 'PT Sans', a sans-serif for clear and accessible text.
- Headline font: 'Playfair', a modern serif to give an elegant and fashionable feel.
- Code font: 'Source Code Pro' for code snippets.
- Incorporate smooth Framer Motion animations for fluid transitions throughout the application.
- Implement an aviation-themed loading screen with progress bar and system indicators to enhance user engagement.
- Animated flight paths with moving planes in the background to reinforce the aviation theme.
- AviationCursor - Use an custom cursor with an animated plane icon and trailing effects. This effect can be toggled.
